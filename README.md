# Sudoku Solver

A web app to solve your hardest sudokus!
Try it [here](https://zivkaplan.github.io/Sudoku-Solver/)

## Features:

-   Backtracking sudoku solving algorithm
-   responsive
-   Dark/light theme toggle
-   input validation
-   control with keyboard

## Project contains:

-   HTML, CSS (Grid, Bootstrap), JS
